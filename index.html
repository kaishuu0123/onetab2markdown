OneTab2Markdown

OneTab の URL の読み込み/書き出しからコピーしたテキストをドメイン別に振り分けて markdown 化するツールです

主な機能

  • OneTab形式のテキストをMarkdownリンクに変換
  • ドメインごとに自動整理・グループ化
  • ドメインでソート、またはそのまま変換も可能
  • ブラウザ上で完結（データは保存されません）
  • 日本語・英語の2言語対応

使い方

  1. OneTabで「URLの読み込み/書き出し」を開く
  2. 「すべてのタブとウィンドウを書き出し」をクリック
  3. コピーしたテキストを貼り付け
  4. 変換モードを選択（そのまま/ソート/グループ化）
  5. 「変換」ボタンをクリック
  6. 生成されたMarkdownをコピー
